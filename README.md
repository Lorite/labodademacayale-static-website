# La Boda de Macayale

A beautiful, password-protected wedding website hosted on Coolify.

## Features

- Responsive design inspired by [Blisend](https://blisend.com/en/~/cUEKg1og24Hez1zr)
- Simple password protection for guests
- Hosted on personal server via Coolify

## Project Structure

```
src/               # Source files (HTML, CSS, JS)
public/            # Static assets
scripts/           # Utility scripts
```

## Development

To be set up based on chosen tech stack.

## Deployment

Deploy to Coolify on your personal server.

## Security Notes

- `.gitignore` excludes private photos, family pictures, and sensitive banking information
- Ensure `.env` files with credentials are never committed
- Use environment variables for API keys and passwords
